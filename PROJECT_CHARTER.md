# Project charter — Power BI portfolio

## North star

Grow the Nordic Boardroom Power BI portfolio: gold → semantic model → PBIR pages → Orbit sync, with Python (or notebook) prep where reports need scored/feature tables — publishable demos that read executive-ready.

## Success checks

- [ ] Portfolio smoke passes (`py -3 scripts/jarvis_smoke.py`)
- [ ] Each featured report folder keeps a valid `.pbip` entry point
- [ ] New/changed gold or Python prep is documented in the report README
- [ ] Architecture docs under `docs/architecture/` stay truthful after pipeline changes
